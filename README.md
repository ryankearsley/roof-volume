# Roof Volume Calculator

A simple, free web app to calculate the enclosed volume of a roof. Hosted on GitHub Pages — no server required.

## Supported Roof Types

| Type | Formula |
|------|---------|
| **Gable** | V = ½ × Length × Width × Height |
| **Hip** | V = W × H × (3L − W) / 6 |
| **Flat** | V = Length × Width × Height |

## Features

- 🏠 Three roof types: Gable, Hip, Flat
- 📐 Labelled 3D diagrams for each roof type
- 🔁 Metric (m³) and Imperial (ft³) with instant conversion
- 📱 Responsive — works on mobile and desktop
- ⚡ No dependencies, single HTML file

## Usage

Open `index.html` in any browser, or visit the GitHub Pages URL.

## Deploying to GitHub Pages

1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Set source to the `main` branch, root folder
4. Your app will be live at `https://<username>.github.io/roof-volume/`
